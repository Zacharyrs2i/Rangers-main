# Rangers Site

Static site for the Rangers Discord.

- **Live Join button** uses your permanent invite.
- **Member / Online counts** fetched from Discord public invite endpoint (no token).
- **Server widget** shows online members (make sure it's enabled in Discord Server Settings → Integrations → Widget).

## Local preview

Open `index.html` directly in a browser or use any static server.

## Deploy

Upload this folder to a **public GitHub repository**, then enable **GitHub Pages** (Settings → Pages → Deploy from a branch → `main` / `/root`).
